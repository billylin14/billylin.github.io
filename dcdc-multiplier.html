    • Descriptions

    We designed a DC-DC voltage doubler to mock the implementation of a voltage converter in an eletronic device. Instead of using transformers, we implemented our doubler using a ring oscillator and a few capacitors and diodes. The ring oscillator was built using a NAND gate from CMOS, and it converts the DC input to an oscillating AC input. The AC input was then fed to a full-wave voltage quadrupler circuit, and the output was carefully probed at the node where it outputs only double of the original input voltage.
